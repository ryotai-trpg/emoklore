# キャラクターシート
キャラクターシートで現在使える主な機能です．現状共鳴者用のもののみですが，NPCや怪異用のものもいずれ実装したいとは思っています．

::: warning
共鳴者用キャラクターシートも，武器などのアイテム・ダメージの適用などの便利な自動化は現状ないです．実装する構想はあります．

フォントサイズ6以上にしたり，ライトモードにしたりなどは表示が崩れます
:::


## プレイモード
モードはウィンドウ右上の人に鍵マークがついているアイコンで切り替えられます．
### ダイスロール
技能タブの中にある，技能名やベース技能名をクリックするとダイスロールがされます．

共鳴判定はヘッダーの♾️のアイコンから行えます．

:::info
ショートカットキーの組み合わせで，手動でダイスボーナスなどを修正できる機能はアップデートで実装する予定
:::

### HP, MP, 共鳴値の管理
現在直接入力で変更できるのみです．

## 編集モード
能力値・技能・プロフィールなどを編集できます．正直まだ操作性はよくないです．
