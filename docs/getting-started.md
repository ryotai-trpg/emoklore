# はじめる
## これは何？
オンラインセッションツールFoundry Virtual Tabletop向け，エモクロアTRPGの非公式システム実装です．

プレイにはルールブックが必要です．[公式サイト](https://emoklore.dicetous.com)で無料で読むことができます．

現在はアルファ版段階で，必要な機能を洗い出している最中ですが，工夫すればセッションにも使えると思います．

## インストール
「ゲームシステムを入手」メニューを開き「URLを指定」にて、以下のURLを入力しインストールしてください．
```
https://github.com/ryotai-trpg/emoklore/releases/latest/download/system.json
```
v13のみの対応です．v12以前への対応はしません．
